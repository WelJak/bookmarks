Projekt witryny internetowej obslugujacej logowanie, rejestracje, dodawanie/usuwanie zdjec, 
upvotowanie zdjec, followanie uzytkownikow, wyswietlanie aktywnosci uzytkownikow

Site project which allows user to register, log in/out, add/delete images, upvote/revert images,
follow another users, see another users activity feed
